# Mentoria Hub — платформа развития для школьников

## Что это?

Платформа, где школьники могут находить олимпиады и хакатоны, проходить мини-курсы, получать персональные рекомендации и отслеживать прогресс.

## Быстрый старт

1. Активируйте окружение: `source venv/bin/activate`
2. Установите зависимости: `pip install django pillow`
3. Подготовьте базу данных: `python manage.py migrate`
4. Создайте суперпользователя: `python manage.py createsuperuser`
5. Загрузите тестовые данные: `python manage.py shell < scripts/load_data.py`
6. Запустите сервер: `python manage.py runserver`
7. Откройте http://127.0.0.1:8000/

## Основные возможности

**Для учеников:**
- Система XP и уровней (каждые 100 XP = +1 уровень, +20 XP за урок)
- Счётчик ежедневной серии (streak)
- Курсы с линейным прохождением уроков (как Duolingo)
- Избранное для олимпиад, хакатонов, конкурсов, стажировок и грантов
- Рекомендации на основе выбранных интересов (Бизнес, STEM, Английский, Поступление)
- Поиск и фильтрация возможностей по категориям

**После регистрации:**
- Онбординг с выбором интересов
- Персонализированная лента рекомендаций
- Отслеживание дедлайнов

## Технологии

Backend: Django 6.0, SQLite, Django ORM
Frontend: HTML5, Tailwind CSS, JavaScript, Font Awesome
Дизайн: полностью адаптивный (мобильные устройства, планшеты, десктопы)

## Структура проекта

- `mentoria/` — основное приложение (модели, вьюхи, шаблоны, сервисы)
- `config/` — настройки Django
- `manage.py` — управление проектом
- `db.sqlite3` — база данных

## Полезные команды

- `python manage.py makemigrations` — создать миграции
- `python manage.py migrate` — применить миграции
- `python manage.py createsuperuser` — создать админа
- `python manage.py runserver` — запустить сервер
- `python manage.py shell` — открыть Django shell

## Админ-панель

Доступна по адресу `/admin/`. Позволяет управлять курсами, уроками, возможностями, новостями и просматривать профили пользователей.

## Безопасность

Реализована CSRF-защита, хеширование паролей, защита от SQL-инъекций.
